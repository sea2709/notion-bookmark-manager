Setup Guide

  1. Go to notion.so/my-integrations and create a new integration. Copy the Internal Integration Token.
  2. Open or create a Notion database with these columns:
    • Title — Title property (built-in)
    • URL — URL property type
    • Tags — Multi-select property type
    • Notes — Text property type
    • Date Added — Date property type
  3. Click ... → Connections in your database and connect the integration you created.
  4. Copy your database ID from the URL:
    notion.so/workspace/[database-id]?v=...
